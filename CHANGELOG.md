# Changelog

All notable changes to this project will be documented in this file.

## v0.1.0 - 2026-04-02

Initial open source release.

### Added

- Conversion from Outlook for Mac `.olm` archives or extracted archive folders
- Mail export to `MBOX`
- Mail export to `EML`
- Contact export to `VCF`
- Calendar export to `ICS`
- Task export to `ICS` with `VTODO`
- Account separation and nested folder preservation
- CLI packaging via `olm-converter`
- Test suite and GitHub Actions workflow

### Notes

- Built around the real Outlook for Mac archive structure
- Notes are not exported in this release
